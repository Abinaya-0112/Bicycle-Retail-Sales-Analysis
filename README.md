# Bicycle-Retail-Sales-Analysis
Designed an end-to-end data analytics project focused on bicycle retail sales. Leveraged Excel, MySQL, Python, and Power BI to clean data, perform business analysis, and visualize key performance metrics through interactive dashboards.
•Consolidate and query sales data stored across multiple related database tables.
•Analyse sales performance by store, product, category, brand, and staff.
•Segment customers using the RFM (Recency, Frequency, Monetary) model.
•Design an interactive Power BI dashboard to present business insights.
